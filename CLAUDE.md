# Claude Project Instructions

Required sequence for every task:
1. Understand the impacted system
2. Confirm requirement and acceptance criteria
3. Analyze risks and security
4. Propose implementation plan
5. Implement
6. Test
7. Validate UI in browser with Playwright if applicable
8. Retest until correct

Never claim success without evidence.
Never skip security review.
Never skip browser validation for UI-facing changes.
